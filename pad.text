Використай команду git status, щоб перевірити статус локального репозиторію
Введи команду git add --all , щоб додати всі зміни під контроль Git
Повтори команду git status, щоб переконатися, що зміни під контролем
Введи команду git commit -m “add index file”, щоб зберегти зміни в історії
Введи команду git push, щоб відправити зміни на GitHub
